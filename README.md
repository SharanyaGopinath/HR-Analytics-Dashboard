# HR-Analytics-Dashboard
HR Analytics Dashboard using PowerBi
HR Analytics Dashboards are powerful tools that leverage data to optimize various human resource functions. These dashboards integrate and visualize key HR metrics, offering insights into employee performance, recruitment, retention, and overall workforce management. By centralizing data from multiple sources, they enable HR professionals to monitor trends, identify issues, and make data-driven decisions.

## Aim and objective
The primary aim of the HR Analytics dashboard was to explore employee attrition trends within the organization. By harnessing data, the goal was to uncover patterns and determinants of attrition rates. This project had two key objectives: providing insights into current attrition and assisting management in taking proactive steps to mitigate these rates. 


## About the dashboard
Built using Power BI, the HR analytics dashboard offers a detailed view of essential metrics to help organizations tackle HR-related issues.

The dashboard highlights crucial KPIs such as Total Employees, Active Employees, Attrition, Attrition Rate, and Average Age. These insights empower HR managers to take proactive actions to retain top talent, manage attrition, and foster a more productive work environment.With the capability to drill down into specific departments and teams, the dashboard provides detailed insights into workforce performance. This enables managers to identify trends and make data-driven decisions to enhance employee engagement, reduce turnover, and boost productivity.
